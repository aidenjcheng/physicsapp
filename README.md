# Physics Timer App

## App Features

| Feature                           | Description                                                                                  | Demo Video                  |
| --------------------------------- | -------------------------------------------------------------------------------------------- | --------------------------- |
| **Find the exact change in time** | Measure the exact time difference between two points in a video. Perfect for physics labs.   | `timestamp_edited.mp4`      |
| **Move frame by frame**           | Move frame by frame to find the exact time difference with auto-detected FPS.                | `frame_by_frame_edited.mp4` |
| **Quickly change videos**         | Use the top upload button to quickly change videos.                                          | `change_video_edited.mp4`   |
| **Check previous measurements**   | See the calculated time difference and access your measurement history from previous videos. | `check_history_edited.mp4`  |
